randify

Randify

Randomize your existing code with randify. Evaluate how your code reacts to randomly distributed inputs and evaluate the probability distributions of your code outputs. Randify performs Monte Carlo simulations to estimate probability density distributions of your outputs. Designed for simple syntax.

Full API documentation at https://jonasnebl.github.io/randify/randify.html

Installation

Install randify from PyPI:

$ pip install randify

Quick guide

Randify works on functions with any number and type of input and output arguments. In this quick example we have a function y1, x2 = f(x1, x2) returning the sum x1 + x2 and the product x1 * x2 of two arguments x1 and x2. For more in-depth examples check-out the Jupyter-Notebooks in examples.

def f(x1, x2):
    return x1 + x2, x1 * x2

x1 = 1
x2 = 2
y1, x2 = f(x1, x2)

Now we want to evaluate how random inputs x1 and x2 influence the results y1and y2 using randify. We can do this using two steps.

  1. Define x1and/or x2as a RandomVariable. For defining the RandomVariable you need to pass a function that generates random samples of this RandomVariables. In this example, we use functions from numpy's random module, but you can also define custom functions (see the examples for that).
  2. Call the randify function wrapper with the RandomVariables as arguments.
from randify import randify, RandomVariable
import numpy as np

def f(x1, x2):
    return x1 + x2, x1 * x2

x1 = RandomVariable(np.random.normal, loc=0, scale=1)
x2 = RandomVariable(np.random.uniform, low=-1, high=1)
y1, y2 = randify(f)(x1, x2)

y1and y2 are now also RandomVariables. You can calculate the resulting statistical measure like expected value or variance. To display the estimated probability distributions, you can use randify's plot_pdf function:

print(f"E[y1] = {y1.expected_value}")
print(f"Var[y1] = {y1.variance}")

from randify import plot_pdf
plot_pdf(x1=x1, x2=x2, y1=y1, y2=y2)

10    :param foo: Function to evaluate.
11    :param N: Number of iterations for Monte Carlo simulation.
12        Default: -1 for automatic number of iterations.
13    :param duration: Approximate (very rough approximate)
14        duration of the simulation in seconds.
15        Used to determine N if N=-1.
16        No effect if N!=-1 or one of the RandomVariables provides samples.
17
18    :param verbose: If True, information about the simulation is printed.
19    :return: function that allows RandomVariables as input and returns RandomVariables as output
20    """

22    def inner(*args, **kwargs):
23        start_total = perf_counter()
24
25        initial_arguments = list(args) + list(kwargs.values())
26        arguments = initial_arguments.copy()  # altered arguments for each iteration
27
28        # check which input variables are RandomVariables
29        random_variable_indices = []
30        for count, arg in enumerate(initial_arguments):
31            if isinstance(arg, RandomVariable):
32                random_variable_indices.append(count)
33
34        # evalute foo once to determine return structure and execution time
35        for random_variable_index in random_variable_indices:
36            arguments[random_variable_index] = initial_arguments[random_variable_index].sample()
37        start = perf_counter()
38        y = foo(*arguments)
39        duration_foo = perf_counter() - start
40        if duration_foo <= 1e-3:  # perform more accurate time measurement for fast functions
41            start = perf_counter()
42            for i in range(9):
43                y = foo(*arguments)
44            duration_foo += perf_counter() - start
45        if N == -1:  # automatic number of iterations
46            N_samples = int(duration / duration_foo)
47        else:

49        returns_tuple = isinstance(y, tuple)
50        if returns_tuple:

53        else:

55            result_samples = []
56

61                random_variable_index
62            ]._return_N_samples(N_samples)

65        for i in range(N_samples):
66            # build function arguments
67            for random_variable_index in random_variable_indices:
68                arguments[random_variable_index] = input_samples[random_variable_index][i]
69            y = foo(*arguments)
70            if returns_tuple:
71                for j in range(N_return):
72                    result_samples[j].append(y[j])
73            else:
74                result_samples.append(y)
75
76        if verbose:
77            print(f"Randify: {N_samples} samples evaluated in {perf_counter() - start_total:.3f}s.")
78
79        if returns_tuple:
80            return tuple(RandomVariable(result_samples[i]) for i in range(N_return))
81        else:

 96    def _return_N_samples(self, N):
 97        """
 98        Returns a list of N samples of the RandomVariable. If more than N samples are available,
 99        N samples are randomly selected. If less than N samples are available,
100        the samples are extended to the number N. Instead of sample(),
101        this function may change the number of self._samples and is for interal use only.
102        :param N: Number of samples
103        :return: list of N samples of the RandomVariable.
104        """
105        if len(self._samples) == N:
106            return self._samples
107
108        elif len(self._samples) < N and hasattr(self, "generator_func"):
109            # new samples from generator function
110            self._samples += self._return_N_new_samples_from_generator_func(N - len(self._samples))
111
112            # delete cached properties based on samples
113            # because more samples are available for more accurate statistical measures
114            if hasattr(self, "expected_value"):
115                del self.expected_value
116            if hasattr(self, "variance"):
117                del self.variance
118            if hasattr(self, "skewness"):
119                del self.skewness
120            if hasattr(self, "kurtosis"):
121                del self.kurtosis
122
123            return self._samples
124
125        elif len(self._samples) > N:
126            selected_indices = np.random.choice(len(self._samples), size=N, replace=True)
127            return [self._samples[i] for i in selected_indices]  # don't reduce number of samples
128
129        elif len(self._samples) < N:
130            selected_indices = np.random.choice(len(self._samples), size=N, replace=True)
131            self._samples = [self._samples[i] for i in selected_indices]
132            return self._samples

 63    """
 64    Calculate the probability density function of one or multiple RandomVariables.
 65    Based on a kernel density estimation using KDEpy.
 66    Examples:
 67    .. code-block:: python
 68        x1 = RandomVariable(np.random.normal, loc=0, scale=1)
 69        x2 = RandomVariable(np.random.normal, loc=0, scale=1)
 70        x3 = RandomVariable(np.random.multivariate_normal, mean=np.zeros(2,), cov=np.eye(2))
 71        pdf(x1)(0) # will give the marginal pdf of x1 evaluated at x1=0.
 72        pdf(x1, x2)(0,0) # will give the joint pdf of x1 and x2 evaluated at x1=0, x2=0.
 73        pdf(x3)([0,0]) # will give the pdf of x3 evaluated at x3=[0,0].
 74        pdf(x1, x3)(0,[0,0]) # will give the joint pdf of x1 and x3 evaluated at x1=0, x3=[0,0].
 75    The function allows multiple values at once if they are stacked along the 0-th axis:
 76    .. code-block:: python
 77        pdf(x1, x3)([0,1], [[0,0], [1,1]]) # will give the pdf at x1=0, x3=[0,0] and x1=1, x3=[1,1].

Calculate the probability density function of one or multiple RandomVariables. Based on a kernel density estimation using KDEpy. Examples:

x1 = RandomVariable(np.random.normal, loc=0, scale=1)
x2 = RandomVariable(np.random.normal, loc=0, scale=1)
x3 = RandomVariable(np.random.multivariate_normal, mean=np.zeros(2,), cov=np.eye(2))
pdf(x1)(0) # will give the marginal pdf of x1 evaluated at x1=0.
pdf(x1, x2)(0,0) # will give the joint pdf of x1 and x2 evaluated at x1=0, x2=0.
pdf(x3)([0,0]) # will give the pdf of x3 evaluated at x3=[0,0].
pdf(x1, x3)(0,[0,0]) # will give the joint pdf of x1 and x3 evaluated at x1=0, x3=[0,0].

The function allows multiple values at once if they are stacked along the 0-th axis:

pdf(x1, x3)([0,1], [[0,0], [1,1]]) # will give the pdf at x1=0, x3=[0,0] and x1=1, x3=[1,1].
Parameters
  • *args: RandomVariable objects to calculate the joint probability density function of.
  • kernel: Kernel for the kernel density estimation. Default "gaussian".
  • bandwidth: Bandwidth for the kernel density estimation. Default None. None will estimate the bandwidth automatically.
Returns

Probability density function at given input.
